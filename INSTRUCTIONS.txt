MOGGEDNYC — ONLY 3 THINGS YOU HAVE TO DO YOURSELF
=====================================================

1) API KEY (Anthropic website — I cannot do this for you)
   In this folder, create a file named exactly:  .env
   Put ONE line inside (replace with your real key):
   ANTHROPIC_API_KEY=sk-ant-api03-xxxxxxxx

   Save the file. Never send your key in chat to anyone.

2) FIRST TIME — make the launcher executable (once)
   Open Terminal, paste this line, press Enter:
   chmod +x ~/Desktop/game/start.command

3) RUN THE APP
   Terminal:
   cd ~/Desktop/game
   npm start

   Read the line it prints:  moggednyc → http://localhost:XXXX
   Open THAT exact address in your browser (port may be 8081 if 8080 is busy).

Claude chat needs BOTH the .env file AND npm start (or start.command).
If something breaks, read the black Terminal window — errors print there.
