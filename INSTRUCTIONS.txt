MOGGEDNYC — ONLY 3 THINGS YOU HAVE TO DO YOURSELF
=====================================================

1) FIRST TIME — make the launcher executable (once)
   Open Terminal, paste this line, press Enter:
   chmod +x ~/Desktop/game/start.command

2) RUN THE APP
   Terminal:
   cd ~/Desktop/game
   npm start

   Read the line it prints:  moggednyc → http://localhost:XXXX
   Open THAT exact address in your browser (port may be 8081 if 8080 is busy).

If something breaks, read the black Terminal window — errors print there.
